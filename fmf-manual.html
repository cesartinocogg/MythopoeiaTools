🗺️ Map for Mythopoeia · User Manual

Map for Mythopoeia is a dedicated cartography tool designed for worldbuilders, GMs, and creatives. It provides a full-featured canvas for drawing, layering, annotating, and exporting high-quality maps. Built on Fabric.js, it delivers a smooth, interactive experience with support for brushes, images, text labels, and a complete layer system.

Everything is stored in a portable .fmf file, making it easy to save, load, and share your projects.

📐 Map Size

Choose from 1K, 2K, or 4K resolution – ideal for everything from quick sketches to detailed regional maps.

🖌️ Drawing Tools

Brush, eraser, image placement, and text labels. Full color control and adjustable brush sizes.

📚 Layer System

Organise your work with multiple layers. Toggle visibility, reorder, and edit independently.

🖼️ Image Gallery

Upload and manage reference images. Drag and drop or double-click to place them on the canvas.

📤 Export Options

Export your map as PNG, JPG, or SVG – perfect for printing, sharing, or further editing.

💾 Native Format

Save your project as a compressed .fmf file. All layers, objects, and images are preserved.

💡 Key concept: The canvas is your map. Use layers to separate elements (terrain, landmarks, labels, notes). The active layer receives all new objects. The property panel lets you tweak selected items without leaving the workspace.

All images and data are embedded in the project file. Save, load, and share with a single .fmf file.

🖥️ Interface Overview

The interface is divided into a sidebar (left) and the canvas (main view). The sidebar can be collapsed with the ☰ button.

Sidebar

  • Map Size – switch between 1K, 2K, or 4K resolution. Changing size clears the canvas (confirmation required).
  • Tools – Select, Brush, Eraser, Image, Text, and Color Picker (Dripper).
  • Brush Options – adjust size (4–120px) and color for brush strokes.
  • Image Gallery – upload images, drag them onto the canvas, or double-click to place them at the centre.
  • Text Tool – enter a label and click Add to place it on the active layer.
  • Layers – create, delete, toggle visibility, and set the active layer.
  • Properties Panel – appears when an object is selected. Edit name, dimensions, color, or text content.
  • Export – download the current map as PNG, JPG, or SVG.
  • Project – save your work as .fmf or load a previous project.
  • Activity Log – shows recent actions for reference.

Canvas

  • The canvas displays your map at the chosen resolution.
  • A gold border marks the map boundary – objects outside this border are clipped.
  • Zoom: scroll to zoom in/out. The zoom level is displayed in the bottom-right corner.
  • Pan: hold Alt and drag to pan around the canvas.
  • Controls: ＋ (zoom in), － (zoom out), ⊡ (fit to view).
🖱️ Navigation: Scroll to zoom • Alt+drag to pan • Double‑click image in gallery to place it • Drag and drop images from gallery to canvas.

🛠️ Tools

Select (↖️)

Default mode. Click any object to select it. The Properties Panel appears on the right side of the sidebar, allowing you to edit:

  • For images: label/name, width, height (maintains aspect ratio).
  • For text: text content and color.

Selected objects can be moved, resized, or rotated. Press 🗑️ Remove to delete the selected object.

Brush (🖌️)

Freehand drawing tool. Adjust Size (4–120px) and Color in the brush options. Strokes are added to the active layer and can be erased using the Eraser tool.

  • Strokes are non‑selectable and cannot be moved or resized.
  • Each stroke is stored as a path with its own color and size.

Eraser (🧹)

Removes brush strokes from the active layer. Adjust the Size slider to control the eraser diameter.

  • Only brush strokes on the active layer are erased.
  • Images and text are not affected by the eraser.

Image (🖼️)

Opens the Image Gallery panel. Upload images using the + Add Image button.

  • Drag & drop: drag a thumbnail from the gallery onto the canvas to place it at the cursor position.
  • Double‑click: places the image at the centre of the map.
  • Images are inserted on the active layer and can be selected, moved, and resized.

Text (📝)

Adds a text label to the canvas. Type your text in the input field and click Add (or press Enter).

  • Text uses the Cinzel font family for a fantasy aesthetic.
  • Text color, content, and position can be edited via the Properties Panel when selected.
  • Text is placed on the active layer.

Color Picker (🧪 Dripper)

Click anywhere on the canvas to sample a color. The picked color is applied to the brush, and the tool automatically switches back to Brush mode.

  • Useful for matching existing map colors.
  • The color is stored in the brush color picker for further use.
💡 Tip: The active layer is highlighted in the layer list. All new objects (brush strokes, images, text) are added to the active layer.

📚 Layers

Layers are a powerful way to organise your map. Each layer can contain multiple objects, and you can toggle visibility, reorder, and edit independently.

Managing Layers

  • Active layer – selected via the dropdown or by clicking a layer in the list. All new objects are added to this layer.
  • Visibility – click the checkbox next to a layer name to show/hide it. Hidden layers are not exported.
  • Delete – click ✕ on a layer to remove it and all its objects. At least one layer must remain.
  • Create – click + New, enter a name, and click Create. The new layer becomes active.

Layer Behavior

  • Objects on a layer inherit the layer's visibility.
  • When you delete a layer, all objects on that layer are removed from the canvas.
  • The Base layer is created by default and cannot be renamed (but can be deleted if other layers exist).
📌 Pro tip: Use layers to separate terrain, labels, landmarks, and notes. This makes editing and exporting much more flexible.

📤 Export & Project Management

Export

Export your map as a high-quality image file. The export captures the entire map at its native resolution.

  • PNG – lossless, supports transparency. Best for digital use.
  • JPG – smaller file size, suitable for sharing and printing.
  • SVG – vector format. Scalable and editable in vector software. Note: brush strokes are exported as vector paths.

The export includes all visible layers. Hidden layers are excluded.

Project File (.fmf)

The native file format is a compressed JSON archive with the .fmf extension. It stores:

  • Map size (width/height).
  • All layers (names, visibility, order).
  • All objects (brush strokes, images, text) with their properties (position, color, size, content).
  • All images embedded as base64 data URLs.

Because images are embedded, the file can be large, but it is completely self‑contained.

Save & Load

  • 💾 Save .fmf – compresses and downloads the current project as a .fmf file.
  • 📂 Load – opens a file picker to load a previously saved .fmf project.

Loading a project replaces the current canvas. Make sure to save any unsaved work before loading.

⚙️ Advanced Features

Activity Log

The log at the bottom of the sidebar tracks your actions – layer changes, tool usage, exports, saves, and errors. Useful for debugging and keeping track of your workflow.

Keyboard Shortcuts

  • Scroll – zoom in/out on the canvas.
  • Alt + drag – pan the canvas.
  • Enter – when typing text, adds the label to the canvas.
  • Double‑click on a gallery image – places it at the map centre.

Color Picker (Dripper)

The 🧪 Dripper tool samples any pixel on the canvas and applies the color to the brush. After sampling, the tool automatically switches back to Brush mode so you can start drawing immediately.

Properties Panel

When an object is selected with the Select tool, the Properties Panel appears. It provides context‑sensitive controls:

  • Image – edit the label/name, and adjust width/height (aspect ratio is locked).
  • Text – edit the text content and the fill color.
  • Brush strokes – are not selectable, so no properties are shown.

Canvas Constraints

All objects are constrained to the map boundaries. You cannot drag an object outside the map area. This ensures your map stays clean and export-ready.

Performance Tips

  • Larger map sizes (4K) require more memory. Use 2K or 1K for faster performance during sketching.
  • Keep the number of brush strokes reasonable – each stroke is a separate path object.
  • Use layers to hide content you're not actively working on – this improves rendering speed.
🔄 Remember: The .fmf file is your project. Save often, and keep backups of important maps.

🗺️ Map for Mythopoeia · Manual de Usuario

Map for Mythopoeia es una herramienta de cartografía diseñada para creadores de mundos, directores de juego y artistas. Ofrece un lienzo completo para dibujar, aplicar capas, anotar y exportar mapas de alta calidad. Construido sobre Fabric.js, proporciona una experiencia fluida e interactiva con pinceles, imágenes, etiquetas de texto y un sistema completo de capas.

Todo se almacena en un archivo .fmf portátil, lo que facilita guardar, cargar y compartir tus proyectos.

📐 Tamaño del Mapa

Elige entre 1K, 2K o 4K – ideal desde bocetos rápidos hasta mapas regionales detallados.

🖌️ Herramientas de Dibujo

Pincel, borrador, colocación de imágenes y etiquetas de texto. Control total de color y tamaño.

📚 Sistema de Capas

Organiza tu trabajo con múltiples capas. Alterna visibilidad, reordena y edita de forma independiente.

🖼️ Galería de Imágenes

Sube y gestiona imágenes de referencia. Arrastra o haz doble clic para colocarlas en el lienzo.

📤 Opciones de Exportación

Exporta tu mapa como PNG, JPG o SVG – perfecto para imprimir, compartir o editar.

💾 Formato Nativo

Guarda tu proyecto como un archivo .fmf comprimido. Todas las capas, objetos e imágenes se conservan.

💡 Concepto clave: El lienzo es tu mapa. Usa capas para separar elementos (terreno, puntos de interés, etiquetas, notas). La capa activa recibe todos los objetos nuevos. El panel de propiedades te permite ajustar los elementos seleccionados sin salir del espacio de trabajo.

Todas las imágenes y datos están incrustados en el archivo del proyecto. Guarda, carga y comparte con un solo archivo .fmf.

🖥️ Visión general de la interfaz

La interfaz se divide en una barra lateral (izquierda) y el lienzo (vista principal). La barra lateral se puede ocultar con el botón ☰.

Barra lateral

  • Tamaño del mapa – cambia entre 1K, 2K o 4K. Cambiar el tamaño borra el lienzo (se requiere confirmación).
  • Herramientas – Seleccionar, Pincel, Borrador, Imagen, Texto y Cuentagotas (Selector de color).
  • Opciones del pincel – ajusta el tamaño (4–120px) y el color de los trazos.
  • Galería de imágenes – sube imágenes, arrástralas al lienzo o haz doble clic para colocarlas en el centro.
  • Herramienta de texto – escribe una etiqueta y haz clic en Add para colocarla en la capa activa.
  • Capas – crea, elimina, alterna visibilidad y establece la capa activa.
  • Panel de propiedades – aparece cuando se selecciona un objeto. Edita nombre, dimensiones, color o contenido de texto.
  • Exportar – descarga el mapa actual como PNG, JPG o SVG.
  • Proyecto – guarda tu trabajo como .fmf o carga un proyecto anterior.
  • Registro de actividad – muestra las acciones recientes para referencia.

Lienzo

  • El lienzo muestra tu mapa en la resolución elegida.
  • Un borde dorado marca el límite del mapa – los objetos fuera de este borde se recortan.
  • Zoom: desplázate para acercar/alejar. El nivel de zoom se muestra en la esquina inferior derecha.
  • Panorámica: mantén Alt y arrastra para desplazarte por el lienzo.
  • Controles: ＋ (acercar), － (alejar), ⊡ (ajustar a la vista).
🖱️ Navegación: Desplázate para hacer zoom • Alt+arrastrar para desplazar • Doble clic en una imagen de la galería para colocarla • Arrastra y suelta imágenes desde la galería al lienzo.

🛠️ Herramientas

Seleccionar (↖️)

Modo por defecto. Haz clic en cualquier objeto para seleccionarlo. El Panel de propiedades aparece en la barra lateral, permitiéndote editar:

  • Para imágenes: etiqueta/nombre, ancho, alto (mantiene la relación de aspecto).
  • Para texto: contenido y color.

Los objetos seleccionados se pueden mover, redimensionar o rotar. Presiona 🗑️ Remove para eliminar el objeto seleccionado.

Pincel (🖌️)

Herramienta de dibujo a mano alzada. Ajusta el Tamaño (4–120px) y el Color en las opciones del pincel. Los trazos se añaden a la capa activa y se pueden borrar con la herramienta Borrador.

  • Los trazos no son seleccionables y no se pueden mover ni redimensionar.
  • Cada trazo se almacena como una ruta con su propio color y tamaño.

Borrador (🧹)

Elimina los trazos del pincel en la capa activa. Ajusta el control deslizante de Tamaño para controlar el diámetro del borrador.

  • Solo se borran los trazos del pincel en la capa activa.
  • Las imágenes y el texto no se ven afectados por el borrador.

Imagen (🖼️)

Abre el panel de la Galería de imágenes. Sube imágenes con el botón + Add Image.

  • Arrastrar y soltar: arrastra una miniatura de la galería al lienzo para colocarla en la posición del cursor.
  • Doble clic: coloca la imagen en el centro del mapa.
  • Las imágenes se insertan en la capa activa y se pueden seleccionar, mover y redimensionar.

Texto (📝)

Añade una etiqueta de texto al lienzo. Escribe tu texto en el campo y haz clic en Add (o presiona Enter).

  • El texto utiliza la fuente Cinzel para un estilo fantástico.
  • El color, el contenido y la posición del texto se pueden editar en el Panel de propiedades cuando está seleccionado.
  • El texto se coloca en la capa activa.

Selector de color (🧪 Cuentagotas)

Haz clic en cualquier lugar del lienzo para muestrear un color. El color elegido se aplica al pincel y la herramienta cambia automáticamente al modo Pincel.

  • Útil para igualar los colores existentes del mapa.
  • El color se almacena en el selector de color del pincel para su uso posterior.
💡 Consejo: La capa activa está resaltada en la lista de capas. Todos los objetos nuevos (trazos del pincel, imágenes, texto) se añaden a la capa activa.

📚 Capas

Las capas son una forma potente de organizar tu mapa. Cada capa puede contener múltiples objetos, y puedes alternar la visibilidad, reordenar y editar de forma independiente.

Gestionar capas

  • Capa activa – se selecciona mediante el menú desplegable o haciendo clic en una capa de la lista. Todos los objetos nuevos se añaden a esta capa.
  • Visibilidad – haz clic en la casilla junto al nombre de la capa para mostrar/ocultarla. Las capas ocultas no se exportan.
  • Eliminar – haz clic en ✕ en una capa para eliminarla y todos sus objetos. Debe quedar al menos una capa.
  • Crear – haz clic en + New, introduce un nombre y haz clic en Create. La nueva capa se convierte en la activa.

Comportamiento de las capas

  • Los objetos de una capa heredan la visibilidad de la capa.
  • Cuando eliminas una capa, todos los objetos de esa capa se eliminan del lienzo.
  • La capa Base se crea por defecto y no se puede renombrar (pero se puede eliminar si existen otras capas).
📌 Consejo profesional: Usa capas para separar el terreno, las etiquetas, los puntos de interés y las notas. Esto hace que la edición y la exportación sean mucho más flexibles.

📤 Exportación y gestión de proyectos

Exportar

Exporta tu mapa como un archivo de imagen de alta calidad. La exportación captura todo el mapa en su resolución nativa.

  • PNG – sin pérdida, admite transparencia. Ideal para uso digital.
  • JPG – tamaño de archivo más pequeño, adecuado para compartir e imprimir.
  • SVG – formato vectorial. Escalable y editable en software vectorial. Los trazos del pincel se exportan como rutas vectoriales.

La exportación incluye todas las capas visibles. Las capas ocultas se excluyen.

Archivo de proyecto (.fmf)

El formato nativo es un archivo JSON comprimido con extensión .fmf. Almacena:

  • Tamaño del mapa (ancho/alto).
  • Todas las capas (nombres, visibilidad, orden).
  • Todos los objetos (trazos del pincel, imágenes, texto) con sus propiedades (posición, color, tamaño, contenido).
  • Todas las imágenes incrustadas como URL de datos base64.

Como las imágenes están incrustadas, el archivo puede ser grande, pero es completamente autónomo.

Guardar y cargar

  • 💾 Save .fmf – comprime y descarga el proyecto actual como un archivo .fmf.
  • 📂 Load – abre un selector de archivos para cargar un proyecto .fmf guardado anteriormente.

Cargar un proyecto reemplaza el lienzo actual. Asegúrate de guardar cualquier trabajo no guardado antes de cargar.

⚙️ Funciones avanzadas

Registro de actividad

El registro en la parte inferior de la barra lateral rastrea tus acciones: cambios de capa, uso de herramientas, exportaciones, guardados y errores. Útil para depurar y hacer un seguimiento de tu flujo de trabajo.

Atajos de teclado

  • Scroll – zoom en el lienzo.
  • Alt + arrastrar – desplazar el lienzo.
  • Enter – al escribir texto, añade la etiqueta al lienzo.
  • Doble clic en una imagen de la galería – la coloca en el centro del mapa.

Selector de color (Cuentagotas)

La herramienta 🧪 Cuentagotas muestrea cualquier píxel del lienzo y aplica el color al pincel. Después de muestrear, la herramienta cambia automáticamente al modo Pincel para que puedas empezar a dibujar inmediatamente.

Panel de propiedades

Cuando se selecciona un objeto con la herramienta Seleccionar, aparece el Panel de propiedades. Ofrece controles contextuales:

  • Imagen – edita la etiqueta/nombre y ajusta el ancho/alto (la relación de aspecto está bloqueada).
  • Texto – edita el contenido y el color de relleno.
  • Trazos del pincel – no son seleccionables, por lo que no se muestran propiedades.

Restricciones del lienzo

Todos los objetos están restringidos a los límites del mapa. No puedes arrastrar un objeto fuera del área del mapa. Esto asegura que tu mapa se mantenga limpio y listo para exportar.

Consejos de rendimiento

  • Los mapas más grandes (4K) requieren más memoria. Usa 2K o 1K para un rendimiento más rápido durante el boceto.
  • Mantén un número razonable de trazos del pincel – cada trazo es un objeto de ruta independiente.
  • Usa capas para ocultar el contenido en el que no estás trabajando activamente – esto mejora la velocidad de renderizado.
🔄 Recuerda: El archivo .fmf es tu proyecto. Guarda con frecuencia y mantén copias de seguridad de tus mapas importantes.
